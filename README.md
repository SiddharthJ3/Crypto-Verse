# Crypto Verse

Crypto Verse is a website that provides insights and information regarding different cryptocurrencies. <br>
It shows real-time, accurate, and updated market data using CoinGecko's crypto `API` to the user. 

## Tech Stack 

- ### `ReactJS`
- ### `JavaScript`
- ### `CSS`
